📱 Convert to APK

Option 1 — PWABuilder (Easiest, 5 min)

1. Upload this HTML to netlify.com (drag & drop, free)

2. Go to pwabuilder.com → paste your URL

3. Click Android → Download APK

4. Install on phone (enable Unknown Sources)

Option 2 — Capacitor (Dev)

npm install @capacitor/cli
npx cap init MoneyMgr
npx cap add android
npx cap open android

Final APK size: ~3–4 MB ✅
